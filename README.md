### Sobre mim

Sou desenvolvedor frontend focado em criar experiências digitais limpas, funcionais e com atenção aos detalhes. Atualmente mergulhado em **TypeScript + React**, com o olho já no horizonte do **Angular**.

Acredito que bom código e boa interface andam juntos — não adianta funcionar se não for agradável de usar.

- 🎯 Foco principal em **TypeScript & React**
- 🎨 Apaixonado por **UI/UX** e experiência do usuário
- 📐 Base sólida em **HTML, CSS e JavaScript**
- 🔭 Próximo passo: **Angular**
- ⚡ Gosto de projetos que misturam **design + código**
